2016年博士毕业的时候，爬过当时的工作职位情况。那时收集的数据已经找不到了。

10年后，又到了好奇这个事情的时候啦。Kimi 帮我做了两个检索。大致情况总结：

### 报告一：《毕业生职业路径统计报告》

| 维度 | 核心发现 |
| :--- | :--- |
| **地区差异：学术就业率** | 美欧顶尖项目（MIT/LSE等） **~75%** ；香港新加坡 **~55-60%** ；中国大陆顶尖（北大/清华） **~60-70%** （因校/地域差异大） |
| **学科差异：学术就业率** | 金融学博士 **70%+** （商学院助理教授起薪 ＄ 200K+，是经济系近2倍）；经济学博士 **50-76%** （项目排名差异大） |
| **项目规模差异** | 金融博士项目：每年招**3-5人**（申请者~250）；经济博士项目：每年招**20-30人**（申请者~800） |
| **科技冲击** | Amazon过去5年雇佣 **150+** 经济学博士；学术职位在美国经济博士就业中占比已降至 **~50%** （JEP 2023） |
| **满意度与失业** | 整体失业率仅 **2.1%** （所有博士学科最低）；**69%** 对首份工作"非常/极其满意"；**93%+** 持积极态度 |
| **学术去向分化** | **~15%** 进入 Harvard/MIT 级顶尖校； **~30%** 进入 Top 50-100 研究型大学； **~20%** 最终离开学术界 |
| **金融博士双轨制** | 学术界：商学院教授薪酬溢价显著（助理教授 ＄200K+ vs 经济系  ＄100-130K）；业界：对冲基金/投行/金融科技，出路更精准 |
| **欧洲特色** | LSE 75%学术就业，**11%** 央行/ **7%** 国际组织（区位独特优势）；博士后文化更普遍（1-3年缓冲） |
| **香港新加坡** | 学术与业界并重（**55-60%** vs **45-40%** ）；金融科技中心提供数据科学等新型岗位；本地博士学术天花板为亚洲区域高校 |
| **中国大陆** | 顶尖985学术就业率 **60-70%** ；**22.6%** 选择博士后作为"缓冲带"；财经类高校博士后比例显著更低 |
| **主要业界去向** | 科技巨头（Amazon/Google/Uber）> 金融（对冲基金/投行）> 咨询（NERA等经济咨询）> 国际组织（IMF/世行）> 央行/政府 |
| **placement质量** | MIT 2020-2025年：76%学术，其中 **32%** 进入 Top 15 经济系/Top 10 商学院；LSE 学术 placement 全球第7（RePEc排名） |

### 报告二：《毕业十年后的职业发展报告》

| 维度 | 核心发现 |
| :--- | :--- |
| **收入鸿沟（10年）** | 业界中位年薪从 ＄109K→ ＄**174K**（+60%），学术界从＄85K→ ＄**120K**（+41%），差距从28%扩大到 **60%+** |
| **马太效应：学术界** | Top 5项目10年后学术界收入＄**185K**，Top 20外仅＄**110K**，差距**1.68倍**；业界差距仅**1.19倍**（＄208K vs ＄175K） |
| **非Top项目学术生存率** | 排名50-100项目10年后留学术圈仅 **35-45%** ，100以外仅 **20-30%** ；Top 20项目为 **65-70%** |
| **离开学术圈后的去向** | 排名50-100项目毕业生：转业界25-30%，转adjunct/community college 15-20%，完全离开领域10-15% |
| **Tenure成功率：Top 50** | 整体Tenure获得率 **~76%** （第5-6年50%，第7-8年18%，延迟8%，失败离开 **24%** ） |
| **Tenure成功率：非Top** | 排名50-100项目10年后留学术圈 **35-45%** ，100以外 **20-30%** ；10年内"失败"概率 **55-80%** |
| **Tenure后发表衰减** | Tenure后2年发表量降 **30%** ，"homerun"论文降30%；Tenure十年后"homerun"率比当年降 **~50%** |
| **性别差距：获Tenure** | 男性11年内获Tenure率 **~75%** ，女性仅 **~61%** ，差距**14.3%** |
| **性别差距：升正教授** | Tenure后7年内升正教授：男性 **~52%** ，女性仅 **~25%** ，差距**26.6%** ；女性正教授比例欧洲仅13%、美国15% |
| **业界晋升（10年）** | **~65%** 晋升中高级；科技巨头Principal Economist ＄200-350K，对冲基金Senior Quant/PM **＄250K-＄5M+** |
| **金融博士 vs 经济博士** | 金融博士商学院教授\$300-500K（经济系仅\$180-280K）；金融博士学术就业率 **75%** vs 经济博士 **55%** |
| **中国特色：非升即走** | Top 985高校**40-50%** 助理教授 **6年内未晋升即离开**；海归博士显著优于本土 |
| **中国特色：收入分化** | 顶尖985+长江学者教授年薪 **100-200万+** ，普通省属教授仅 **20-35万**，差距 **5-10倍** |
| **香港新加坡** | 学术天花板为亚洲区域正教授（非全球Top 50）；金融博士对冲基金PM 10年后 **＄600K-＄3.8M** |
| **欧洲** | 央行/国际组织提供"中间地带"路径（€100-150K）；政策路径无Tenure压力、工作生活平衡更好 |
